# Bella Vista Bistro - Restaurant Website

## 🍽️ Project Overview

Bella Vista Bistro is a modern, responsive restaurant website designed to showcase the establishment's culinary offerings, ambiance, and brand identity. The site features an elegant, user-friendly interface that adapts seamlessly across all devices while maintaining a sophisticated aesthetic that matches the restaurant's upscale dining experience.

## 🌟 Features

### Interactive Navigation

- Transparent to frosted-glass header effect on scroll
- Smooth scrolling to sections
- Mobile-responsive hamburger menu
- Active section highlighting

### Dynamic Menu System

- Category-based filtering system
- Smooth animations for item transitions
- Responsive grid layout
- Clear pricing and descriptions

### Visual Engagement

- Parallax scrolling effects
- Fade-in animations on scroll
- Image hover effects
- High-quality food photography

### User Interaction

- Interactive contact form with validation
- Real-time form feedback
- Social media integration
- Responsive button and link hover states

### Responsive Design

- Mobile-first approach
- Fluid typography
- Flexible grid systems
- Optimized images for different screen sizes

## 🚀 Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/bella-vista-bistro.git
   ```

2. Navigate to the project directory:

   ```bash
   cd bella-vista-bistro
   ```

3. Open index.html in your browser or use a local server:

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve
   ```

4. View the site at `http://localhost:8000`
